Das Projekt soll irgendwann ein einfaches Spiel werden, wo es jedem Nutzer möglich ist eigene Level zu kreieren.
Man kann sich anmelden (Datenbank und Server vorrausgesetzt) und mit dem Map Editor eigene Karten erstellen.

Das Projekt habe ich urspünglich nur mit JavaScript angefangen und später auf JQuery umgestellt.
Dabei habe ich erkannt wie schnell, dass Projekt mit JQuery umsetzbar ist und wie viel Code man spart.
